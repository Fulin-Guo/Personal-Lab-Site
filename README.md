# Personal Lab Site

A monochrome, future-editorial personal homepage for presenting resume, selected works, technical writing, and project updates.

## Content

- Homepage with strong typography and sharp visual identity
- Resume page
- Selected works / project cases
- Technical blog powered by Markdown
- Changelog for site updates
- Chinese / English language switch

## Tech Stack

- Astro
- TypeScript
- Astro Content Collections
- CSS
- pnpm
