This homepage is designed as a personal lab instead of a generic portfolio.

The visual system uses a small set of primitives:

- oversized typography
- repeated text fields
- sharp dividers
- numbered content blocks
- restrained accent color

The goal is to make the site memorable while keeping resume, projects, and technical writing readable.
